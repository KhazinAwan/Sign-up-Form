# Volleyball District Sign-Up Form

A sign-up page for **Volleyball District**, a fictional platform where local volleyball teams can organize matches, manage players, and connect with other teams.

This project was created as part of **The Odin Project** curriculum and focuses on HTML forms, CSS layout techniques, custom fonts, form validation states, and Flexbox.

## Features

* Two-column desktop layout
* Full-height background image sidebar
* Custom self-hosted font
* Form validation using HTML attributes
* Invalid input styling with `:user-invalid`
* Focus styling using `:focus`
* Password length validation
* Custom branding and logo section
* Accessible form labels linked to inputs

## Validation Features

The form currently validates:

* Required fields
* Email format
* Minimum password length

Password matching is not implemented, as this requires JavaScript and is covered later in The Odin Project curriculum.
